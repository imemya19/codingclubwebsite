CCSOC event-IIT Kanpur Network series and the prerequisites to get started

CCSOC event-IIT Kanpur Network series and the prerequisites to get started

Posted by: Shubham Singhal
Posted on: 17 April 2017

IIT Kanpur will be offering an online MOOC course on Cyber Security during June-July 2017.

Prerequisites:
-Very strong programming background with knowledge of run-time or execution environment, usage of debuggers, and knowledge of shared libraries or dynamically linked libraries
-Basic knowledge of x86 assembly language or similar assembly languages will be assumed. -Basic knowledge of Operating Systems especially memory management, virtual memory etc. will be assumed.
-Basic network protocols such as TCP/IP, DNS, routing etc.
-Prior knowledge of a scripting language such as shell scripting, Perl, python and/or Ruby
-Knowledge of Javascript, PHP or other web programming
for more information visit ::https://www.securitymooc.in/
For practice -
1) Go through any vulnerability we want to explore like sql, crypto, xss, cookies related vulnerabilities i.e web,reverse engineering,forensic,cryptography
2) As there is no defined course for any of the above mentioned topic you will need to google as these field are highly dynamic so visit blogs and more importantly also see how to fix the vulnerability.
3) Go to ctftime.org ,here you will be able to sharpen your skills.
